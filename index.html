Project Description

LIRI is like iPhone's SIRI. However, while SIRI is a Speech Interpretation and Recognition Interface, LIRI is a Language Interpretation and Recognition Interface. LIRI will be a command line node app that takes in parameters and gives you back data. Clone it and try it out!

Project Keywords: App Development, Backend Development, Command Line Interface, CLI

Check my twitter
